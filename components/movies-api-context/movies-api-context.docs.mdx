### Overview

This component is a concrete example of the [api-context-base component](https://bit.dev/learn-bit-react/context/api-context-base), with pre-populated
baseurl and api key.

These can both be overriden, for instance when wanting to test out a different data environment for the movies api (pre-production, laboratory, etc).

Example usage:

Provider side
``` js
app.js

import { MovieApiContextProvider } from '@learn-bit-react/context.context.movies-api-context';

...
    <MovieApiContextProvider baseUrl={config.baseUrl} apiKey={config.apiKey}>
        <App />
    </ MovieApiContextProvider>

```

Consumer side:
```
import { UseMovieContext } from '@learn-bit-react/context.context.movies-api-context';

// build api call, declaring parameters to be populated into the url
const apiCallConfig = (searchStr: string): MoviesContextResource => ({
      params: {
        s: searchStr
      }
    });

const [data, getDataApiCall] = UseMovieContext(apiCallConfig)

```
